# MPU-6050
A full Kicad module of pysical project about the MPU-6050 is a popular, low-cost 6-axis motion tracking sensor. It combines a 3-axis accelerometer and a 3-axis gyroscope. It uses standard I2C communication and is widely used in DIY projects like drones, balancing robots, and digital levels.
# MPU-6050 KiCad Hardware Design

A full KiCad hardware design project for the **MPU-6050**—a popular, low-cost 6-axis Motion Tracking Device combining a 3-axis accelerometer and a 3-axis gyroscope.

---

## 📌 Features & Overview

* **Sensor:** MPU-6050 (3-Axis Accelerometer + 3-Axis Gyroscope)
* **Communication Protocol:** Standard $I^2C$ Interface
* **Applications:** DIY Electronics, Drones, Self-Balancing Robots, Digital Levels, and Motion-Tracking Devices
* **EDA Tool:** KiCad PCB Design Suite
